This page provides an introduction to the process flows that govern our applications and services.

## Key Concepts

Before diving into the specifics, it's important to understand some key concepts related to process flows:

- **Actors**: The individuals or systems that interact with the process.
- **Triggers**: Events that initiate the process.
- **Outcomes**: The expected results of the process.
